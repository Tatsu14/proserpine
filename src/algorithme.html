Calcul du Score

L'Eco-Score Proserpine repose sur une transparence totale. Il est calculé en deux étapes majeures :

1. Le Socle ACV (80% de la note)

Nous analysons 6 indicateurs environnementaux clés : Carbone, Consommation d'eau, Impact sur la biodiversité du sol, Eutrophisation, Qualité de l'air et Ressources fossiles.

2. Les Modérateurs (Bonus/Malus)

Le score de base est ajusté selon des critères spécifiques :

  • Emballage : Bonus pour le vrac (+10) ou le verre réutilisable, malus pour le plastique (-10) ou le non-recyclable (-20).
  • Transport : Malus important pour le fret aérien (-30).
  • Agriculture : Bonus pour le BIO (+10) ou le régénératif (+15).
  • NOVA : Pénalité pour les produits ultra-transformés (-20).
